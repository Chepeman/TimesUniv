    WINDOW APPLICATION : TimesUniv Project Overview
___________________________________________________________________________

The Wintempla Wizard has created the TimesUniv application

Wintempla is located in four areas of visual studio:
 
    File > New > Add Project
    Project > Add New Item
    Tools > Wintempla GUI Manajer
    View > Toolbox



___________________________________________________________________________

The Wintempla Wizard has created the following resources:

TimesUniv.rc
    This files includes a list of all the icons, bitmaps, and cursors

Resource.h
    This is the standard header file, which defines new resource IDs.
    Microsoft Visual C++ reads and updates this file.

TimesUniv.ico
    This is an icon file, which is used as the application's icon (32x32).
    This icon is included by the main resource file TimesUniv.rc.


___________________________________________________________________________

Other standard files:

StdAfx.h, StdAfx.cpp
    These files are used to build a precompiled header (PCH) file
    named TimesUniv.pch and a precompiled types file named StdAfx.obj.
